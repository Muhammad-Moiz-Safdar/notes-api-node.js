# Notes API — Raw Node.js

A REST API built with pure Node.js (no frameworks) as part of my backend learning journey.

## Endpoints

| Method | URL | Description |
|---|---|---|
| GET | /api/notes | Get all notes |
| GET | /api/notes/:id | Get note by id |
| POST | /api/notes | Create a note |
| PUT | /api/notes/:id | Update a note |
| DELETE | /api/notes/:id | Delete a note |

## Tech used
- Node.js (http, url, fs modules)
- No frameworks — raw Node only

## Run locally
npm install
npm run dev
